# 项目路线图

[English](./ROADMAP_EN.md) | 简体中文

## 当前版本 (v1.0.10)

- 基本磁盘信息获取功能
- 跨平台支持 (Windows, macOS, Linux)
- TypeScript 类型定义
- Node.js 12+ 兼容性

## 短期计划 (v1.1.0)

### 新功能

- [ ] 获取所有磁盘信息，而不仅是主磁盘
- [ ] 添加磁盘使用率计算
- [ ] 支持自定义磁盘路径查询
- [ ] 添加磁盘类型识别 (HDD/SSD/网络驱动器等)

### 改进

- [ ] 优化错误处理和错误信息
- [ ] 增强文档和示例代码
- [ ] 添加更多平台支持 (FreeBSD, Android)
- [ ] 性能优化

## 中期计划 (v1.2.0)

### 新功能

- [ ] 磁盘性能监控 (读写速度)
- [ ] 磁盘健康状态查询
- [ ] 文件系统信息获取
- [ ] 磁盘分区信息

### 改进

- [ ] 添加异步 API
- [ ] 流式 API 支持大磁盘
- [ ] 缓存机制减少系统调用
- [ ] 更多单元测试和集成测试

## 长期计划 (v2.0.0)

### 新功能

- [ ] 完整的磁盘管理工具集
- [ ] 磁盘空间分析和清理建议
- [ ] 磁盘活动历史记录
- [ ] 磁盘性能基准测试工具

### 架构改进

- [ ] 插件系统支持扩展
- [ ] 微服务架构支持
- [ ] Web 界面和可视化
- [ ] REST API 服务

## 贡献优先级

1. **高优先级**
   - 获取所有磁盘信息
   - 优化错误处理
   - 增强文档

2. **中优先级**
   - 磁盘性能监控
   - 异步 API
   - 更多平台支持

3. **低优先级**
   - 插件系统
   - Web 界面
   - REST API 服务

## 发布计划

- **v1.1.0**: 预计 2025年第四季度
- **v1.2.0**: 预计 2026年第一季度
- **v2.0.0**: 预计 2026年第二季度

_注意：以上计划可能会根据社区反馈和实际开发进度进行调整_
